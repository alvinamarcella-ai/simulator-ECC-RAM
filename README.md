# simulator-ECC-RAM
Program ini mensimulasikan ECC RAM dengan Hamming Code (72,64). Data 64 bit dikodekan menjadi 72 bit melalui delapan bit paritas untuk mendukung deteksi dan koreksi kesalahan. Program mencakup proses encoding, penyimpanan, penyisipan error, perhitungan sindrom, serta koreksi kesalahan satu bit.
